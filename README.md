# UPOR JS
###### UPOR is a word in Bengali Language means Up. 
###### UPOR Js is a tiny Vanilla Javascript library using it,
###### you can create a Back To Top button very EASILY with a few lines of code.

### HTML
```html
<button>Upor Jao</button>
```
### CSS
```css
button{
    position: fixed;
    bottom: 2%;
    right: 2%;
    			opacity: 0;
			visibility: hidden;
    z-index: 9999;
}
```
